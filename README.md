# TTC-Delay-Data-Analysis


Author: Kelvin Nguyen

This project analyzes TTC subway delay data using Python, pandas, and matplotlib.

Project Objectives:
- Identify the most common categories of TTC subway delays
- Measure the overall impact of delays using total delay minutes
- Evaluate preventability of major passenger-related delay causes

Tools Used:
- Python
- Pandas
- Matplotlib

Key Findings:
- Passenger-related delays were the most frequent.
- Infrastructure-related delays created the largest total impact.
- Disorderly patrons and unauthorized track access were identified as high-impact and potentially preventable delay causes.
